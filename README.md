## DocuMentor 
Which means "Docu" refers to Documents & "Mentor" refers to Mentor. Mentoring the other people using the documents and the resources <br> which will be useful.
<br> In terms it is a student community web application in which there are communities in which students will be sharing all the resources, documents,etc.. as well as participate in the discussions, questions raised, etc..

## Tech Stack 
- Html
- CSS
- Javascript
- Django
- JQuery
- Figma
- Pythonanywhere (temporary deployment purpose)

## Summary 
This is a college project, in which we tried solving the problem of having interaction with other students academically by building an online web application through which they can interact & improve in their academics.
